# **Анализ рынка недвижимости Санкт-Петербурга и ЛО**

### **Задачи проекта**

- определить средние сроки продажи недвижимости и ключевые рыночные характеристики
- выявить сезонные тенденции активности рынка
- сравнить особенности спроса в Санкт-Петербурге и районах Ленинградской области

### **Описание проекта**

Проведён анализ данных агентства недвижимости для определения рыночных закономерностей и оптимизации бизнес-стратегии.

### **Ключевые направления анализа**

- **сегментация объявлений** по скорости продажи
- **анализ зависимости** между характеристиками объекта и временем его реализации
- **исследование сезонности** публикации и закрытия предложений
- **динамика цен** за квадратный метр в различных сегментах
- **сравнительный анализ** активности рынка по населённым пунктам

### **Основные выводы**

- выявлены **типы недвижимости** с максимально коротким и длинным сроком продажи
- определены **ключевые параметры**, влияющие на скорость сделки (цена, площадь, локация)
- обнаружены **сезонные пики** активности публикации и продаж
- установлены **различия в динамике** рынка Санкт-Петербурга и районов ЛО

### **Рекомендации**

1. скорректировать ценовую политику для разных сегментов на основе скорости продаж
2. планировать маркетинговые активности с учётом сезонных пиков спроса
3. дифференцировать стратегию работы с объектами в СПб и районах ЛО
4. оптимизировать сроки выставления объектов в зависимости от их характеристик
