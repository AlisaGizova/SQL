# Анализ данных для агентства недвижимости

**Задачи проекта:** Проанализировать данные об объявлениях по недвижимости для определения ключевых рыночных характеристик: сроков активности предложений, сезонных тенденций и особенностей рынка в Санкт-Петербурге и Ленинградской области.

**Описание проекта:** Проведен анализ данных агентства недвижимости. Определены сегменты с наиболее короткими и длинными сроками активности объявлений, выявлены зависимости между параметрами недвижимости и временем её продажи. Исследована сезонность публикации и снятия объявлений, проанализирована динамика стоимости квадратного метра. Проведён сравнительный анализ активности рынка по населенным пунктам Ленинградской области. На основе результатов сформулированы выводы и практические рекомендации для планирования бизнес-стратегии.
